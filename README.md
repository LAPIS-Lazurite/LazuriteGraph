# LazuriteGraph
LazuriteGraph is Java application to show graph from sensor data.

# 外部JARファイルのインストール
ソースコードを使用するためには次のライブラリをJAVAのフォルダにインストールしてください。
動作確認済のJARファイルは、libフォルダに保存してあります。
  jcommon-1.0.23.jar
  jfreechart-1.0.19.jar
  RXTXcomm.jar

インストール先のフォルダ
  jre/lib/ext/

# Win32用
シリアルを使用するためにDLLをJAVAのフォルダにコピーする必要があります。
  rxtxSerial.dll
  rxtxPararell.dll
インストール先のフォルダ
  jre/bin/
